# reactivepy

A reactive Python kernel.

# Dependencies

- `ipython>=4.0.0`
- `jupyter_client`
- `tornado>=4.0`
- `ipykernel>=4.8`

# Install

```bash
git clone https://github.com/jupytercalpoly/reactivepy.git

# install dependencies

pip install .
```

# Develop

```bash
git clone https://github.com/jupytercalpoly/reactivepy.git

# install dependencies

pip install -e .
```
